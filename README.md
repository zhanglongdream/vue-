# spinner-vue


```
  npm i spinner-vue --save

```

 
  如果你安装了淘宝镜像 可以使用cnpm i spinner-vue --save <br/>


  如果没有安装建议安装一个毕竟国内的影响心情 [点击安装](https://npm.taobao.org/)

 或者

```
  import spinner from 'spinner-vue'

  <spinner type="snake"></spinner>
  或者这样写
  <spinner :type="0"></spinner>


  type 的值有        'snake',
                    'double',
                    'triple',

  :type  的值有  0 ，1，2 


```
