#toast-vue 


```
  npm i toast-vue --save

```

 
  如果你安装了淘宝镜像 可以使用cnpm i toast-vue --save <br/>


  如果没有安装建议安装一个毕竟国内的影响心情 [点击安装](https://npm.taobao.org/)

 或者

```
import toast from 'toast-vue'

toast('faasassa')

toast({
  message:'fsdsa',
  position:'top',
  duration:1000
})

message 提示信息

position 提示位置

duration 显示消失的速度
```
